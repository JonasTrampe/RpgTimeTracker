Handout
=======

A Markdown-authored text handout the GM can push to players, shown in
its own overlay window on the Handout tab - separate from the image/
video gallery and the map, so sending one doesn't interrupt or replace
whatever's currently displayed.

Enter a title and content (with a Markdown edit/preview toggle) and
click "Send to players". "Retract" closes it for everyone. A newly
connecting or reconnecting client catches up on whatever handout is
currently shown, the same way it catches up on the current image/video.

Only a single handout draft is kept (no reusable library yet) - the
title and content are lost if the app restarts before sending.
