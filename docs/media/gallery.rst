Slideshow / Gallery
===================

Every sent item is kept instead of replaced by the next - the GM and
each player browse independently through everything shown so far (arrow
keys or clicks, purely local). The GM can "highlight" an item for
everyone (a suggested jump) or remove it from the gallery; an optional
seconds-per-image setting auto-advances everyone (videos are never
interrupted). Event media (see :doc:`trigger-media`) briefly interrupts
the gallery without joining it, resuming once the trigger resets.
