Timer
=====

Any number of countdown timers with a progress bar and remaining-time
display. Run on game time (not real time) and react correctly to time
jumps in both directions while running.

An attached image/video/sound is sent to all players automatically on
expiry (see :doc:`/media/trigger-media`); manually resetting the timer
closes it again if still shown. Optionally shows an advance-warning
banner to the GM before it fires (see :doc:`/ui/index`).
