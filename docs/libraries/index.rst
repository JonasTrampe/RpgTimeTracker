Libraries
=========

Media, Sound, Music, Maps, and Characters all follow the same shape: add
items ahead of time, they persist across restarts, and each can be backed
up via Export/Import into its own folder. Items can live in the always-
present Shared Library or be scoped to a single :doc:`Session
</sessions/index>`.
