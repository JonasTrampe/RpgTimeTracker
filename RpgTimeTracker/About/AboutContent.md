# RPG Zeitverfolgung — SL-Steuerung

Steuert Spielzeit, Timer, Alarme und Medien für die Spielrunde und sendet sie
live an verbundene Spieler-Clients im lokalen Netzwerk.

- **Chronometer**: Spielzeit vor-/zurückspulen, Timer/Wecker/Intervalle mit
  eigenem Sound und optionalem Auslöse-Medium.
- **Bibliothek**: Bilder, Videos und Sounds vorbereiten und per Klick an alle
  Spieler senden, inklusive Diashow-Galerie und Lautstärke-/Trim-Einstellungen.
- **Netzwerk**: eingebauter Server mit LAN/mDNS-Erkennung, keine manuelle
  Konfiguration auf Spielerseite nötig.

Läuft komplett lokal im eigenen Netzwerk — es werden keine Daten an externe
Server gesendet.
