# RPG Zeitverfolgung — Spieleranzeige

Zeigt die von der SL gesendete Zeitleiste sowie Bilder, Videos und Sounds auf
einem Spielergerät im lokalen Netzwerk an.

- **Zeitleiste**: aktuelle Spielzeit sowie laufende Timer/Wecker/Intervalle,
  immer synchron zur SL.
- **Medien**: Bilder/Videos öffnen automatisch in einem eigenen Fenster,
  inklusive Diashow-Navigation per Pfeiltasten oder Klick.
- **Verbindung**: verbindet sich per LAN/mDNS-Suche oder manueller
  Host-Adresse mit der SL-Steuerung.

Läuft komplett lokal im eigenen Netzwerk — es werden keine Daten an externe
Server gesendet.
